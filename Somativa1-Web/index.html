Atividade Formativa - Fundamentos de Programação Web

HTML

Definição e estruturação do Site

HTML(HyperText Markup Language) é a base de toda página web. Define estrutura o conteúdo, como títulos, parágrafos, imagens e links. Pense nele como o esqueleto do site — sem HTML, nada aparece na tela.

Estrutura

CSS

Apresentação e visual do Site

CSS (Cascading Style Sheets) cuida da aparência. Ele é responsável por cores, fontes, espaçamentos e a disposição dos elementos na página. Com o CSS, um site simples se transforma em algo visualmente agradável e moderno.

Estilo

JavaScript

Manipulação e interatividade do Site

JavaScript (JS) traz interatividade. Ele permite que botões funcionem, animações aconteçam e dados sejam atualizados sem recarregar a página. Com JS, a experiência do usuário se torna dinâmica e envolvente.
