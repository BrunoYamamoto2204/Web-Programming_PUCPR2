Atividade Formativa - Fundamentos de Programação Web

Este site foi desenvolvido para a Atividade Somativa 1 da disciplina de Fundamentos de Programação Web. A aplicação é inteiramente baseada no lado cliente (front-end), utilizando ferramentas como HTML, CSS e Javascript, tecnologias abordadas durante as unidades nas semanas 1 a 4. Por meio deste projeto pude aplicar os conceitos estruturais, visuais e interativos, integrando as três linguagens em uma solução completa.
Na página inicial há a introdução e definição das 3 linguagens, seguido pela página de login, com formatação e coleta de dados utilizando Javascript, assim como está página sobre, que contém também imagens. Todas as páginas incluem HTML e CSS.
